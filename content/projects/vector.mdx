### abstract
A C++ library to perform basic operations on vectors such as :
- vector addition
- vector subtraction 
- dot product
- cross product
- distance between two points (2D/3D)

### background
Last sem, I took a course on Object Oriented Programming in C++ at my university. So I created this library with the objective of revising :
- Constructors
- operator overloading
- template functions and classes
- friend functions
- C++ build system using Makefiles

### How to use?
To use the library, copy the [vector.h](https://github.com/MahendraDani/vector/blob/main/src/inc/vector.h) header file in your project and include it in your source files.

#include<iostream>
#include "vector.h"

int main(){
  Vector<int> vectors[5];

  for(int i=0;i<5;i++){
    Vector<int> temp(i,i,i);
    vectors[i] = temp;
  }

  // print all vectors
  for(auto x :vectors){
    std::cout << x << std::endl;
  }
  // OUTPUT
  /*
    (0,0,0)
    (1,1,1)
    (2,2,2)
    (3,3,3)
    (4,4,4)
   */
  return 0;
}
